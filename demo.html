üéØ What is PeekInTheCloud?

PeekInTheCloud is a privacy-preserving, browser-based tool that helps you analyze cloud credentials and visualize accessible services. All processing happens in your browser - no credentials are sent to any server.

üîê Security-First

Client-side processing only. No credential storage or transmission to external servers.

‚òÅÔ∏è Multi-Cloud Support

Supports AWS, Azure, and GCP with comprehensive service scanning.

üìä Visual Results

Clean, modern interface showing accessible services with detailed permissions.

üì§ Export Capabilities

Copy results to clipboard or download JSON for further analysis.

üöÄ Quick Start

Get started with PeekInTheCloud in just a few steps:

  1. Open the application: Click the button below to launch PeekInTheCloud
  2. Select a cloud provider: Choose from AWS, Azure, or GCP
  3. Enter credentials: Fill in the required fields for your chosen provider
  4. Scan services: Click "Scan Credentials" to analyze accessible services
  5. Review results: View the visual grid of accessible services

üìä Understanding Results

Service Status

  • Accessible: Service is available and accessible
  • Inaccessible: Service exists but access is denied
  • Error: Network or authentication issues

Permission Levels

  • No Access: Cannot access the service
  • List Only: Can list resources but not read details
  • Read Access: Can read resource details
  • Write Access: Can modify resources
  • Full Access: Complete control over resources

üõ†Ô∏è Troubleshooting

Common Issues

  • "SDK not loaded": Check internet connection and refresh the page
  • "Invalid credentials": Verify credential format and required fields
  • "Network error": Check internet connectivity and firewall settings
  • "Access denied": Verify credentials are correct and haven't expired

Open browser developer tools (F12) to see detailed error messages and API call logs.
